# System Tests

Системные тесты для проверки нефункциональных требований.

## Категории

- `performance/` - Тесты производительности
- `security/` - Тесты безопасности
- `platform/` - Кроссплатформенные тесты
- `e2e/` - End-to-end сценарии

## Метрики

- Время выполнения
- Использование памяти
- CPU нагрузка
- Стабильность
