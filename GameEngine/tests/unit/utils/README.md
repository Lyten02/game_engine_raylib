# Utility Tests

Тесты вспомогательных функций и утилит.

## Категории

- Математические функции
- Строковые операции
- Файловые операции
- Хелперы и конвертеры

## Примеры

- `test_math_utils.cpp` - Математические утилиты
- `test_string_helpers.py` - Строковые функции
- `test_file_utils.py` - Работа с файлами
